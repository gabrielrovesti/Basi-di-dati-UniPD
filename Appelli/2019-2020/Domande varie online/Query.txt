Query 2:

CREATE VIEW SET_VINTI_PER_NAZIONE AS
SELECT Nazione, setvinti
FROM GIOCATORE, PARTITA
WHERE Codice = CodVincitore AND anno = "2010"
UNION
SELECT Nazione, SETPERSI AS setvinti
FROM GIOCATORE, PARTITA
WHERE Codice = CodPerdente AND anno = "2010";

Una volta che la vista di cui sopra e' stata creata, si puo' utilizzare la query di sotto:

SELECT NAZIONE, AVG(SETVINTI)
FROM SET_VINTI_PER_NAZIONE
GROUP BY NAZIONE;


Query 3: 

CREATE VIEW SET_VINTI_PER_NAZIONE AS
SELECT Nazione, COUNT(*) AS TOTALE
FROM GIOCATORE, PARTITA
WHERE Codice = CodVincitore AND anno = "2019"
GROUP BY NAZIONE;

SELECT NAZIONE
FROM SET_VINTI_PER_NAZIONE
WHERE TOTALE=SELECT MAX(TOTALE) FROM SET_VINTI_PER_NAZIONE;
