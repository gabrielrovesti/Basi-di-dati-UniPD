Risolto dopo varie tribolazioni, per installare PgAdmin4 ho usato questo risultato qui:
https://stackoverflow.com/questions/58239607/pgadmin-package-pgadmin4-has-no-installation-candidate

curl https://www.pgadmin.org/static/packages_pgadmin_org.pub | sudo apt-key add
sudo sh -c 'echo "deb https://ftp.postgresql.org/pub/pgadmin/pgadmin4/apt/focal/ pgadmin4 main" > /etc/apt/sources.list.d/pgadmin4.list && apt update'
sudo apt install pgadmin4

Successivamente, se come me fallite in ogni modo possibile, si consiglia il link
dove usare i due comandi con Docker:
mkdir ~/.pgadmin4  # to store config and stuff
docker run -d --rm --network host -v ~/.pgadmin4:/pgadmin thajeztah/pgadmin4
Poi aprire un browser e digitare 
http://localhost:5050/
E si vede PgAdmin da lì (modalità server)
